What is MyFamilyBenefits?
MyFamilyBenefits is an online system that offers Arizona families an easy and secure way to access information about:
  • Nutrition Assistance (Food Stamps)
  • TANF Cash Assistance
For more information about these programs, visit our Web site.
What Can I Do on MyFamilyBenefits?
View information about:
  • Applications for assistance
  • Benefit status
Report Changes:
  • Address
  • Family Members
  • Income
  • Expenses
Mid Approval Contact
  • Submit Mid Approval Contact Form
How do I register for MyFamilyBenefits?
You can only register for MyFamilyBenefits if you have already applied or are receiving benefits.Click here to apply.

You will need your Case Number and other personal information to match our records. Then, you will need to create an account by entering a User Name, password and e-mail address. If you do not have an e-mail address, you can set one up for FREE from Gmail , Yahoo , or Outlook . Finally, you will need to read and accept the User Agreement. If you need assistance creating an account, click Help and select the Account Creation topic.
